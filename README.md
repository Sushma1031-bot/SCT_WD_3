# Tic-Tac-Toe Web Application

## 📌 Project Overview

This project is an interactive Tic-Tac-Toe Web Application developed using HTML, CSS, and JavaScript. The game allows two players to take turns marking spaces in a 3×3 grid and automatically determines the winner or a draw.

## 🚀 Features

- Two-player gameplay (Player X vs Player O)
- Interactive game board
- Automatic winner detection
- Draw detection
- Turn indicator
- Restart Game functionality
- Responsive design for desktop and mobile devices
- Clean and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🎮 Game Rules

1. The game is played on a 3×3 grid.
2. Player X starts first.
3. Players take turns placing their marks (X or O).
4. The first player to align three marks horizontally, vertically, or diagonally wins.
5. If all cells are filled and no player wins, the game ends in a draw.

## 💡 Functionality

### Player Turns
The application alternates turns between Player X and Player O.

### Winner Detection
The program checks all possible winning combinations after every move.

### Draw Detection
If all cells are occupied without a winner, the game declares a draw.

### Restart Game
The Restart button clears the board and starts a new game.
